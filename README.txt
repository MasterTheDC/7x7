7x7 Guide App
==============

使い方
------
1. この zip を展開します。
2. `index.html` をブラウザで開きます。
3. 外部ゲームで出た形を、このアプリでも選びます。
4. 盤面に表示された推薦位置を見て、ゲーム側に反映します。
5. アプリ側でも `確定` を押して盤面を進めます。

補足
----
- `Undo` で一手戻せます。
- `Reset` で最初からやり直せます。
- `手動補正モード` を有効にすると、各マスを直接 ON/OFF できます。
- 推薦は事前生成済みの方策テーブルを優先し、未知状態では distilled policy で候補を絞った上で value-guided expectimax を回します。

必要環境
--------
- BigInt をサポートする比較的新しいブラウザ
- ネット接続は不要です
